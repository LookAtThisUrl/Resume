A modern, minimal WordPress theme. Decode features the ability to change all the colors in the theme, an elegant sidebar, custom Social Icons built right in, link posts, where the title can be hyperlinked, and Featured Images. It is a mobile first, totally Retina-ready, adaptive theme built with HTML 5 specifications. It can be downloaded on GitHub and in the Theme Directory.

The Decode Reply Tool is the perfect complement to the Decode theme. It allows you to continue the conversation with your readers on Twitter or App.net. It is available for free right now in the Plugin Directory.
